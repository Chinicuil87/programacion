# Títulos.

Para hacer el uso de títulos en Markdown se ocupa el “#” antes de cada título.

Al igual que html se tiene 6 opciones y estas irán cambiando de acuerdo al numero de “#” que se usen.

``` md
# Título h1
```
# Título h1

``` md
## Título h2
```
## Título h2

``` md
### Título h3
```
### Título h3

``` md
#### Título h4
```
#### Título h4

``` md
###### Título h5
```
##### Título h5

``` md
###### Título h6
```
###### Título h6
